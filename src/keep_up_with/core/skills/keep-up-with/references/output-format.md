# Output Format

## Quick FYI / BTW

- 1-4 sentences.
- Main link inline.
- What happened.
- Why now / what changed.
- No headings or source dumps.

## Medium / Deep Thread

```markdown
Background:
Who is involved. Add context for unknown people, startups, repos, papers, or projects.
When it happened. If old, why it matters now.

What happened / what is it:
Plain English. 2-4 short sentences.

What's changed / new:
Prior story state, baseline, or comparable. Omit if fake.

Highlights:
- Highest-value facts only.
- Ordered by importance.

Sources:
- [Source](url) - what it supports.
```

Rules:

- One good story beats several shallow notes.
- Source non-obvious claims.
- Keep thread sources out of the main channel.
- Use visuals only to clarify, verify, or compare.
- Omit empty/artificial sections except `Sources`.
