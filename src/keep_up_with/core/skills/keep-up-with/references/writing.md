# Writing

Voice:

- concise
- factual
- natural
- source-grounded
- uncertainty clear

Avoid:

- generic preambles
- internal narration
- "New detail:"
- "worth watching"
- "promising"
- "interesting"
- "notably"
- "important"
- "significant"
- unsupported judgment

Final pass:

- Point clear immediately?
- Right depth?
- Useful to this user?
- Old facts omitted unless needed?
- Claims sourced?
- Visuals useful?
- No padding?
