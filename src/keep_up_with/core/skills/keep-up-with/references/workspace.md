# Workspace

Medium/deep dives:

```text
threads/
  YYYY-MM-DD-HHMM-slug/
    research/
      notes.md
      artifacts/
    output/
      output.md
      assets/
```

Folder:

- local time: `YYYY-MM-DD-HHMM`
- short lowercase hyphen slug
- example: `2026-06-01-1430-deepmind-genie-3`
- reuse for the same story

`research/`:

- internal legwork
- facts, timeline, comparisons, questions
- source notes
- screenshots, figures, transcripts, frames, tables, excerpts

`output/`:

- camera-ready draft
- assets to post

No workspace for one-off quick FYIs.
