# Reference Message: Iroh 1.0

Iroh 1.0 shipped this week from n0. It is a networking library for apps that need devices to find and talk to each other without making users deal with IP addresses, ports, or router settings.

The simple version: an app can ask Iroh to connect to another device, and Iroh handles the annoying networking part. If a direct connection works, it uses it. If not, it can still get the two sides talking through a fallback path.

This is getting real developer attention: the repo is around 9k GitHub stars and the HN launch thread is active. Worth a look if you build local-first, peer-to-peer, sync, multiplayer, or device-to-device apps.

Source: https://github.com/n0-computer/iroh/releases/tag/v1.0.0
