# Reference Message: OpenWA

OpenWA is a self-hosted WhatsApp automation tool that is trending on GitHub today.

It links to a WhatsApp account the way WhatsApp Web does, then gives you an API and dashboard for sending, receiving, and reacting to messages. Think local/internal WhatsApp automation without building the WhatsApp Web glue yourself.

It is around 9k GitHub stars, with a visible spike this month. The main caveat is simple: this is unofficial WhatsApp Web automation. For serious customer messaging, Meta's official WhatsApp Business platform is still the safer route.

Source: https://github.com/rmyndharis/OpenWA
