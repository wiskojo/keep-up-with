# Reference Message: Zvec

Alibaba's Zvec is getting attention again after a new release this week.

Zvec is a local search component for apps that need RAG or semantic search without running a separate vector database server. The idea is closer to “drop this library into your app” than “operate another Pinecone/Milvus-style service.”

The new release makes it more useful as a complete search layer by combining normal keyword-style search with vector search. The repo is around 10k stars, so this is not just a tiny demo.

Source: https://github.com/alibaba/zvec/releases/tag/v0.5.0
